Visual Studio Code

Tipsy && tricki

  • (Przy instalacji zaznaczyc Open with Code to context menu 2 cbx, register Code as an editor for supperted file types, add to PATH)
  • w pilku ! + tab - tworzy podstawowy format pliku rozpoznajac po rozszerzeniu
  • ctrl + shift + p - okienko do wpisywania poleceń (podpowiada polecenia i skróty)
  • alt + shift + f - autoformatowanie dokumentu lub w okienku poleceń wpisać format document -> autoformatiwawnie
  • view -> toggle word wrap - zawijanie wierszy
  • Rozszerzenie Prettier do formatowania dokumentu. Ctrl + shif + p -> komenda format document. Po instalacji może zgłosić kilka formatterów i wybieramy prettiera
  • Rozszerzenie Live server do automatycznego odpalania strony. Prawy dolny róg przycisk Go Live i strona w przeglądarce będzie odświeżać się autoamtycznie.

Html

Warstwa 1 Frontendu

  • Nazwy plików html bez spacji - zamiast tego myślnik
  • <!DOCTYPE html> zaszłość, kiedys oznaczało się wersje htmla
  • <html lang="pl"> przy np. stronie ze wspomaganiem dla niedowidzących włączaja się odpowiednie funkcje w określonym języku
  • <main></main> główna część dokumentu
  • <section></section> podział na sekcje (zbiór artykułów)
  • <article><article> podział na artykuły
  • <header></header> oraz <footer></footer> - nagłówek i stopka, poza całym dokumentem mogą także być zawarte w <section> i <article>
  • <h1-h6> nagłówki tytułowe, h1 - zgodnie ze sztuką tylko jeden na stronie powinnien się znajdować
  • <p> paragrag/akapit
  • <br/> enter (większy odstęp - stylami)
  • <i> <em> pochylenie
  • <b> <strong> pogrubienie
  • <b> oraz <i> nie mają znaczenia semantycznego - służą do wyróżnienia tekstu
  • <strong> i <em> mają znaczenie semantyczne, strong - nadanie ważności tekstu, em - oznaczenie ironii np.
  • atrybuty html
  • <a href="" target=""><link> link do innej strony, pliku lub sekcji na stronie (sekcji nadać id="", a w <a href odwołać się przez #id_sekcji>
  • <img src="" alt=""/> umieszczenie obrazka, src - adres obrazka, alt - tekst alternatywny jeśli obrazek się nie wyświetli
  • <nav> semantyczne menu
